# To-Do-List

To Do List Website made from the [Odin Project's](https://www.theodinproject.com/) curriculum.

[Live Demo](https://skim2264.github.io/To-Do-List/)

## Built with

- JavaScript
- Webpack
- CSS
- HTML

## Getting started

```
git clone https://github.com/skim2264/To-Do-List.git
cd To-Do-List
npm install
npm run start
```
